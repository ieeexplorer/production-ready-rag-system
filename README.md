# Production-Ready RAG System

A fully optimized Retrieval-Augmented Generation platform designed for enterprise deployment and high-reliability applications. This version incorporates advanced techniques for improved performance, security, and scalability.

## Major Improvements
- Persistent vector storage with ChromaDB
- Hybrid retrieval combining semantic and keyword search with reranking
- Comprehensive RAG evaluation using RAGAS metrics
- Multi-layered guardrails including content moderation and validation
- Enhanced observability with structured logging and metrics
- Optimized Docker setup for production environments
- CI/CD pipelines with automated testing and security checks

## Getting Started

1. Clone and checkout this branch
2. Set up environment variables
3. Install dependencies and run services

The system is now better prepared for real-world usage with focus on robustness and maintainability.
